# Slam Poetry MO — 5. terv: Főoldal Implementation Plan

**Goal:** A főoldal összeállítása a Higgsfield homepage-mockup (`docs/superpowers/design-mockups/01-homepage.png`) blueprintje szerint, a meglévő komponensekből.

**Architecture:** Az `src/pages/index.astro` újraírása: sötét hero (3D `ScrollMic` + nagy display cím + magenta „sticker" akcentus + lead + CTA-k), majd három tartalmi szekció világos háttéren: **Közelgő események** (`EventCard` + `splitEvents`), **Slammerek** (új `SlammerCard` rács), **Friss hírek** (`PostCard`). Minden szekció `SectionHeader`-rel, „összes" linkkel, üres-állapot-biztos. Reszponzív, on-brand.

**Tech:** meglévő helperek: `getEvents`/`splitEvents`, `getSlammers`, `getPosts`; meglévő komponensek: `ScrollMic`, `SectionHeader`, `EventCard`, `PostCard`. Új: `SlammerCard.astro`, `.sticker` CSS.

## Tasks

### Task 1: SlammerCard + sticker stílus
- Create `src/components/SlammerCard.astro` (portré-kártya: 3:4 fotó, név, város; `urlForImage`).
- Append `.sticker` osztály a `src/styles/global.css`-hez: enyhén elforgatott magenta címke, ink szöveg, vastag display betű, kis árnyék.
- Build verifikáció.

### Task 2: Főoldal összeállítása
- Rewrite `src/pages/index.astro`:
  - Hero: `ScrollMic`, `.sticker` „20+ éve élő közösség" jellegű címke, `h1` „Slam Poetry Magyarország", lead, két CTA (`/esemenyek`, `/mi-az-a-slam-poetry`).
  - „Közelgő események": `splitEvents(getEvents).upcoming.slice(0,4)` `EventCard`-okkal; üres állapot szöveg; „Összes esemény →" link.
  - „Slammerek": `getSlammers().slice(0,8)` `SlammerCard` ráccsal; „Összes slammer →".
  - „Friss hírek": `getPosts().slice(0,3)` `PostCard`-okkal; „Összes hír →".
  - Minden szekció `SectionHeader` (label + cím), üres-állapot-biztos.
- Build + vizuális ellenőrzés (Playwright screenshot a mockuphoz hasonlítva).

## Verifikáció
- `npm run build` sikeres (9 oldal), `npm test` zöld.
- A főoldal a sötét heró + 3 szekció struktúrát mutatja; üres datasettel „hamarosan" üres állapotok; a 3D mikrofon csak itt jelenik meg és görgésre mozog.
